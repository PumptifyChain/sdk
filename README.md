**Pumptify** is a Web3 music streaming platform with an integrated crypto wallet. Discover tracks, playlists, and the latest blockchain trends in one place.

## Overview

Pumptify is a decentralized music streaming platform designed for the Web3 era. It combines music discovery with blockchain technology, enabling users to stream tracks, manage digital assets, and interact with the crypto ecosystem in a single interface.

Artists and listeners connect through a transparent and decentralized environment powered by blockchain technology.

## Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/pumptify.git
cd pumptify
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

---

## Environment Variables

Create `.env` file:

```
RPC_URL=your_rpc_url
WALLET_PRIVATE_KEY=your_private_key
API_PORT=3000
```
